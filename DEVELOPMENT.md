# Development

1. Install Node.js via [Node Version Manager (nvm)](https://github.com/nvm-sh/nvm#installing-and-updating):

   ```bash
   nvm install
   ```

1. Enable [Corepack](https://github.com/nodejs/corepack):

   ```bash
   corepack enable
   ```

1. Install dependencies:

   ```bash
   pnpm install
   ```
